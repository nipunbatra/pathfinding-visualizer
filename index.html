Algorithm Info

Breadth-First Search

Explores layer by layer, guaranteeing shortest path in unweighted graphs.

  • Uses Queue (FIFO)
  • Optimal for unweighted graphs
O(V + E)

Depth-First Search

Explores deep before wide. No shortest path guarantee.

  • Uses Stack (LIFO)
  • Lower memory usage
O(V + E)

A* Search

Combines actual cost and heuristic for efficient pathfinding.

  • f(n) = g(n) + h(n)
  • Manhattan distance heuristic
  • Optimal and complete
O(E)

Dijkstra's Algorithm

Always expands node with lowest cost from start.

  • Works with weighted graphs
  • A* with h = 0
O((V+E) log V)

Greedy Best-First

Uses only heuristic to goal. Fast but not optimal.

  • Fast execution
  • No optimality guarantee
O(b^m)

Bidirectional BFS

Searches from both start and end simultaneously.

  • Faster than regular BFS
  • Guarantees shortest path
O(b^(d/2))
